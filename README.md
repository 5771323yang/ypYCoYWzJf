# 项目简介

## 核心功能
- 用户功能模块：首页查看、个人信息查看与修改、农产品信息查询、论坛发布与查看、新闻资讯筛选查看、订单管理
- 专家功能模块：首页查看、论坛管理、聊天回复、个人中心修改与查看
- 管理员功能模块：首页平台系统基本信息查看、用户账号信息管理（新增、修改、删除）、农产品信息管理、论坛交流管理、订单信息管理查询

## 适用场景
适用于农业信息交流与农产品展示平台，为农民、农业专家和管理者提供一个集农产品信息分享、技术交流、订单处理于一体的综合性服务系统。平台通过整合农业资源，促进农业信息流通，提高农产品交易效率，同时为农业专家提供技术支持渠道，帮助解决农业生产中的实际问题。

## 技术特点
基于SSM（Spring+SpringMVC+MyBatis）框架开发，采用B/S架构设计，实现前后端分离。系统模块化设计，支持多角色协同工作，具备良好的扩展性和维护性。
